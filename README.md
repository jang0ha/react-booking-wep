### 페이지구성

- 홈
    - 라우터
        - 홈(/)
        - 룸 전체 보기(rooms)
        - 룸 디테일 (rooms/:id)
        - 대시보드 Outfit사용
- 룸 전체 보기 페이지
    - 라우터 → 룸 디테일 (rooms/:id) 로 이동
    - 필터 기능 →
        - 방 유형
        - 가격 범위
        - 정렬(가격, 신규)
- 룸 디테일 페이지
- 로그인 → 클럭 사용
- 대시보드 → (클럭 로그인 후 이용 /owner)
    - Dashboard (/owner)
    - Add Room (/owner/add-room) → 이미지 업로드,  룸타입, 가격, 에머니티
    - List Room (/owner/list-room)
